# PyVELM v0.26.2

**Release date:** 2026-05-30

Patch release: **workflow sidebar links** under **Workflows → Operations** now
resolve to the correct view module.

---

## Highlights

### Workflow menu hrefs

**Instances**, **Approvals**, and **Tasks** in the admin sidebar were built as
``/web/views/admin/workflow_instance.list`` (and siblings) and returned 404.
Those views live in the **workflow** module; menu entries now use
``view_module="workflow"`` so URLs are ``/web/views/workflow/…``.

After upgrading, run ``pyvelm migrate`` (or **Apps → admin → Sync**) to refresh
stored ``ir.ui.menu`` hrefs. Install the **workflow** module from **Apps** if
those list views are not yet in your database.

---

## Upgrade

```bash
pip install -U pyvelm==0.26.2
pyvelm migrate
```

No schema changes in the framework package for this release.
