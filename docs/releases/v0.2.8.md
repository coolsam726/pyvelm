# pyvelm v0.2.8

**Release date:** 2026-05-23  
**Base module:** `0.18.0` (unchanged)

Symmetric **Many2many** cache invalidation and an updated documentation home page.

---

## Highlights

### Symmetric M2M cache

`Partner.tag_ids` and `Tag.partner_ids` share one junction table. After
v0.2.8, mutating either side drops the stale tuple cache on the other:

- `write({"tag_ids": [...]})` on partners
- `unlink()` on tags or partners with M2M links

### Docs home page

[docs/index.md](../index.md) now lists the current PyPI version, quick-start
paths, a “what’s new” table, and bundled modules (`console`, `vellum`).

---

## Install

```bash
pip install pyvelm==0.2.8
```

## Upgrade from v0.2.7

1. `pip install -U pyvelm` (or `pip install -e .`)
2. Restart app / cron workers

No schema changes.
